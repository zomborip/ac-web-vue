# AC Web *(Vue - Tailwind)*

## Kritérium:

*Node.js >= v21.6*

## Beüzemelés:


- Töltsd le a repót
- A letöltött mappában `npm install`

## Használat:

- Dev HotReload futtatás: `npm run dev`
- Buildelés: `npm run build`

## Build:

- `npm run build`
- Majd a *dist* mappá tartalmát egy webszerverre töltsd fel
